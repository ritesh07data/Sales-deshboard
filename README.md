 # 📊 Excel Sales Dashboard Project

## 🚀 Overview
This project showcases an **interactive Sales Dashboard built using Microsoft Excel** to analyze mobile sales data. The dashboard provides clear insights into sales performance, customer behavior, and transaction trends.

---

## 📌 Key Metrics
- 💰 **Total Sales:** 1M+
- 📦 **Total Quantity Sold:** 47
- 🧾 **Total Transactions:** 9
- 📊 **Average Sales:** 34K

---

## 📈 Dashboard Features
- 📅 Sales Analysis by Month
- 📆 Sales by Day Name
- 📱 Sales by Mobile Model
- ⭐ Customer Rating Distribution
- 💳 Transactions by Payment Method

---

## 🛠️ Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters
- Data Cleaning & Transformation
- Dashboard Design & Formatting

---

## 🎯 Key Insights
- Highest sales observed on **Thursday**
- Most sales generated by **iPhone 12**
- Majority of transactions done via **Cash**
- Customer ratings are mostly positive (4 & 5)
